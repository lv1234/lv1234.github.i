Git学习

Posted on 2018-06-01 | In git

今天学习一下简单的git语法在这里做一下简单的总结，作为将来复习的依据。git commit的作用是在原有的基础将新加入的修改记录下来并生成更新后的目录。可以在改动时注明改动的原因可以做为查找时候的依据，我为了在结构对这样的语法有一个深入的认识现在就是做一些简单的总结，工作区就是我们自己建立的一个文件夹，在这个文件夹内有一个后缀为.git的文件，这个.git就是我们要找的版本库（repository）其中有一个stage称为暂存区或者index可以将工作区的文件通过git add添加到到版本库的暂存区，暂存区中文件可以通过使用git commit加入分支，一般默认建立的分支名称为master,我们来将这个过程分为几步来完成：

1、建立版本库

该如何建立版本库呢？

我们可以通过在自己建立的文件夹中使用git init在文件夹中创建一个.git结尾的文件，这就是我们创建的版本库，在这个版本库中包含内上述的暂存区，分支，我们新建立的文件通过使用git add+文件名都会加入到暂存区（stage/index）通过git commit 加入目录master。

2、在建立了版本库之后我们使用以下几种语法呢？

git commit 使用git commit可以推送至分支，并且可以添加注释。
git log 可以查询我们每次向分支添加的改动时间和改动的原因
git branch +分支名 可以创建新的分支
git checkout +分支名可以实现分支之间的切换
git HEAD 可以退回至上已经更改的未更改

初始列表先浅浅的谈一下我自己的学习对于python的理解就是我在接下来的时间就回忆一下自己对于list理解这样对于自己的提高也是非常的显著。闲话少叙，直接切入正题。

列表的语法表示
Name =[“Jack”,”Mike”,”Billy”,”Sarah”]
可以在外面由中括号包围，其中的数据类型在这里是字符串也可以是其他的类型，使用逗号“，”隔开每一项。
若要访问某一项就需要使用这样的形式Name[0],这一点类似于数组。

列表的方法

append方法：在列表中添加一项可以使用append()方法，还是使用上面的列表Name,可以为其添加一项Name.append(“Jace”)注意这里只能由一个参数。所添加的项会排在列表的末尾。

pop方法：pop方法可以删除列表末尾的项，并将这一项返回。

remove方法：删除特定项如这样使用Name.remove(“Billy”)

insert方法：在列表特定的位置插入数据项Name.insert(1,”Billy”)

extend方法：在将列表进行合并，这里合并不是求交集，若有两个列表Names,Name,可以这样合并 Name.extend(Names)

下面对于列表的使用我可以举出更多实用的例子来充实我自己的学习的观感这样对于我自己在接下来理解是非常的有帮助的。

列表中嵌套列表

def intel(names):
    for each_item in names:
        if isinstance(each_item,list):
            intel(each_item) 
        else:
            print(each_item)

总结

list列表在可以将不同的数据类型的项放在一个列表中，例如这样：
Name =[1,”yes”]这是合法的。
我可以可以操作来向列表中添加特定项：

Name.insert(1,Name[0])
//在这里你会看到将Name的第一项加入了Name这个列表中。

删除特定项也可以像下面这样：
Name.remove(Name[0])
//删除结合了访问某一项的方式
