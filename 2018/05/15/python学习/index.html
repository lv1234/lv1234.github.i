使用Head first python教程开启我的python学习之旅。
在学习了一些简单的函数之后我发现需要重点学习一下split()函数的使用以及参数情况，现在总结如下作为以后的review时的参考。

##语法

`str.split(str="",num=string.count(str))`

##参数

str–分隔符，默认为所有的空字符、换行（\n）、制表符（\t）等。
num–分割次数

##返回值
return分割后的字符串列表，注意这里返回的是一个list列表，列表的例子可以在这里举一个：[‘james’,’jack’,’lion’]这是一个列表。

##实例1
str="Line1-abacdef\nLine2-abc\nLine4-abcd"

print str.split() //取消掉所有的空格符

print str.split(' ',1)

##实例输出结果
['Line1-abcdef','Line2-abc','Line4-abcd']
['Line1-abcdef','\nLine2-abc''Line4-abcd']

##实例2
str="hello boy<[www.baidu.com]>byebye"

str.split("[").split("]")[0].split(".")

'www.baidu.com'

`str.split("[")[1].split("]")[0].split(".")`
//从左往右执行三次
['www','baidu','com']
